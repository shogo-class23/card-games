『Mark-Inscribed Monster』 ゲーム仕様書

１．ゲーム概要
本作は、戦略的な思考とカードのシナジーを重視した対戦型カードゲームである。 プレイヤーと相手（AI）はそれぞれ山札からカードを引き、モンスターを構成・強化・攻撃させることで、相手モンスターを破壊しポイントを獲得する。

**勝利条件**
相手モンスターを撃破して5ポイントを先取する。
または、相手フィールドのモンスターを全滅させる。

**特徴**
モンスターの名前や能力が自動生成ロジックにより多様に変化し、常に新しい戦略が求められる。

２．遊び方とルール

**デッキ構成**
合計30枚のカードで構成。同名カードは2枚まで投入可能。

**ターンの行動**
1. カードを1枚引く
2. モンスターの配置
3. パワー注入
4. 攻撃
5. サポートカードの使用

**ターン終了時**
1. 状態チェック（ダメージ等の判定）を行う。
2. フィールドが時計回りに回転し、次のターンへ移行する。

**制限事項**
*   **攻撃制限**: 各モンスター固有の「必要パワー」を満たす必要がある。パワー不足時は攻撃不可（パワーは次ターン以降へ保持される）。
*   **サポート制限**: サポートカードの使用は1ターンにつき2枚まで。

３．システム上の工夫

**動的な名前生成**
`generateHuntingName` 関数により、サイズ・プロパティを組み合わせた多様な名前を持つモンスターが登場する。

**フィールド回転システム**
`rotateFields` 関数による強制的な位置移動。前衛（攻撃役）と後衛（待機・回復役）の役割が毎ターン入れ替わるため、数ターン先を読んだ配置が不可欠となる。

４．カードの種類と詳細

**【モンスターカード】**
*   攻撃に必要なパワー量を持つ。
*   前衛にいる場合のみ攻撃可能。
*   状態異常を付与する特殊な個体も存在する。

**【状態異常（マイナスパワー）】**
ターン終了時の「状態チェック」段階で効果が発揮される。
*   **アシッド・ポイズン**: 毎ターン1ダメージを与える。
*   **イルネス**: 毎ターン2ダメージを与える強力な病。
*   **スリープ**: 覚醒判定（コイントス）に成功するまで攻撃行動が不能となる。

**【特殊能力モンスター】**
*   **コピー能力**: 相手モンスターの攻撃力を模倣して戦う。
*   **カリイジョウ**: 直接的な打撃ではなく、状態異常の付与に特化。

**【サポートカード】**
*   **ドロー系**: 2枚〜4枚のカードを引き、手札を補充する。
*   **位置操作系**: 「相手呼び出し」や「自分移動」により、フィールド上の配置を入れ替える。
*   **強化・回復系**: パワーの付与や、永続する状態異常を解除する「コンディションヒール」。
*   **その他**: バフ・デバフ、手札の引き直しなどが存在する。
