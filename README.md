# Mark-Inscribed Monsterゲーム仕様書

## 📋 ゲーム概要
本作は、戦略的な思考とカードのシナジーを重視した対戦型カードゲームである。
プレイヤーと相手（AI）はそれぞれ山札からカードを引き、モンスターを構成・強化・攻撃させることで、相手モンスターを破壊しポイントを獲得する。

### 🏆 勝利条件
* 相手モンスターを撃破して **5ポイント** 先取する。
* または、相手フィールドのモンスターを **全滅** させる。

### ✨ 特徴
* モンスターの名前や能力が **自動生成ロジック** により多様に変化。
* 常に新しいモンスターとの出会いがあり、臨機応変な戦略が求められる。

---

## 🎮 遊び方とルール

### 📦 デッキ構成
* 合計 **30枚**。同名カードは **2枚** まで投入可能。

### 🔄 ターンの流れ
1.  **ドロー**: カードを1枚引く
2.  **配置**: モンスターをフィールドへ
3.  **チャージ**: パワーを注入
4.  **攻撃**: 前衛モンスターによるバトル
5.  **サポート**: 魔法カードの使用（1ターン2枚まで）
6.  **終了**: 状態チェック ＆ **フィールド回転**

### ⚠️ 重要な制限事項
* **攻撃制限**: 各モンスターの「必要パワー」を満たす必要がある。
* **サポート制限**: サポートカードの使用は **1ターンにつき2枚** まで。

---

## 🌀 フィールド回転システム
`rotateFields` 関数により、毎ターン陣形が強制的に入れ替わる。
* **前衛**: 攻撃が可能。相手の攻撃対象にもなる。
* **後衛**: 待機・支援・回復・パワーチャージの拠点。
> **戦略の要**: 数ターン後の「回転」を予測した配置が勝利の鍵だぴょん！

---

## 🃏 カードの種類と詳細

### 【モンスターカード】
* **特殊能力持ち**:
    * **コピー（模倣）**: 相手の力を写し取る。
    * **カリイジョウ（異常付与）**: 攻撃時に呪いを刻印する。
    * **ジョウカ（回復）**: 自らの力で味方の状態異常を解除する。

### 【状態異常（マイナスパワー）】
ターン終了時の「状態チェック」段階で効果が発揮される。
*   **アシッド / ポイズン / ダーティ / ブラッド**: 毎ターン1ダメージ。
*   **イルネス / タイヤード**: ターン経過で **ダメージが増加** する。
*   **スリープ**: コイントスに成功するまで行動不能。
*   **パラリシス（麻痺）**: 行動不能状態です。攻撃できません。

### 【サポートカード】
* **ドロー系**: 2〜4枚補充。手札を全て入れ替える「Hand Ex」も強力。
* **位置操作系**: 「相手呼び出し」「自分移動」で回転の運命を変える。
* **特殊魔法**: 
    * **SKIP**: 相手のターンを丸ごと消し去る。
    * **リバース**: 回転方向を逆転させる。
* **回復・強化系**:
    * **コンディションヒール**: 厄介な状態異常を即座に治す。
    * **ブースト / ルーズ**: パワーを操作する。
    * **バフ / デバフ**: 攻撃力を一時的に増減させる。
