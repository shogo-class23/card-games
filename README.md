# Mark-Inscribed Monster（刻印されしモンスター）

## 1. ゲーム概要

トランプの要素（数字やスート）をベースにしつつ、独自の「モンスター名（サイズ＋属性＋種族）」と「属性刻印」を核とした戦略バトルゲームです。

## 2. 現在の遊び方（ルール）

### 先攻・後攻の決定
ルーレットによってターンを決めます。

### モンスターの配置
手札（画面下部）からモンスターを選び、場（スロット）に配置します。モンスター名は自動的に「小さな（サイズ）＋ヒール（属性）＋ラビ（種族）」のように生成されます。

### バトル
自分のターンにモンスターを選択し、相手のモンスターを指定して攻撃します。

### A（攻撃力）
相手のHPを削る値。

### HP
0になるとカードが消滅します。

### 勝利条件
相手のモンスターを全滅させる、または5点先取。

## 3. 工夫点とこだわり

### 動的なネーミング
単なるカード名の他に、3つの要素を組み合わせることで膨大な種類のモンスターを存在させています。

### 情報の階層化
*   左上: サイズ＋属性＋種族（モンスターのアイデンティティ）
*   中央: 属性（戦略的な要となる力）
*   下部: 戦闘ステータス（実数値） 
このように視線の動きに合わせた配置を検討しています。

## 4. 状態異常（カリイジョウ）システム
特定のカード（例：「カリイジョウ」）は、攻撃の代わりに相手に永続的な状態異常を付与する能力を持ちます。この状態異常は、付与されたカードが墓地に行くまで解除されません。

### 現在実装されている状態異常
- **アシッド (Acid)**: 状態チェックのたびに **1** ダメージを受けます。
- **イルネス (Illness)**: 状態チェックのたびに **2** ダメージを受けます。
- **ポイズン (Poison)**: 状態チェックのたびに **1** ダメージを受けます。

## 5. その他の特殊能力を持つモンスター

### カリコピー (Karikopi)
相手の前衛モンスターが持つ攻撃力や特殊能力をコピーして、そのターンだけ使用できる能力を持ちます。

### カリヒール (Kari-Heal)
*   **通常攻撃**: 3ダメージの攻撃を行います。
*   **特殊能力**: 味方の前衛モンスター1体のHPを全回復させます。
*   「通常攻撃」と「特殊能力」は、それぞれ1ターンに1回ずつ使用可能です。
