『Mark-Inscribed Monster』ゲーム仕様書
１．ゲーム概要
｛状態異常モンスターなどの特殊なモンスターと回転するフィールドで戦い抜け｝
本作は、戦略的な思考とカードのシナジーを重視した対戦型カードゲームである。プレイヤーと相手（AI）はそれぞれ山札からカードを引き、モンスターを配置・強化・攻撃させることで、相手モンスターを破壊しポイントを獲得する。
•	タイトル: Mark-Inscribed Monster
•	勝利条件: 相手モンスターを撃破して5ポイントを先取するか、相手フィールドのモンスターを全滅させること。
•	特徴: モンスターの名称や能力が自動生成ロジックにより多種多様に変化し、常に新しい戦術が求められる。
２．遊び方とルール
•	デッキ構成: 合計30枚のカードで構成。同名カードは2枚まで投入可能。
基本の流れ
•	先行・後攻を決定する
•	両者がカードを配置する
•	各ターンで以下の行動を行う
1.	カードを1枚引く
2.	モンスターの配置
3.	パワー注入
4.	攻撃
5.	サポートカードの使用
•	ターン終了後、状態チェックを行う
•	フィールドが時計回りに回転し、次のターンへ移行する
•	制限事項:
o	攻撃には各モンスター固有の「必要パワー」を満たす必要がある。パワーが不足している場合、攻撃はできない。（パワーはターンを跨いで保持される。）
o	サポートカードの使用は1ターンにつき2枚までに制限されている。
•	攻撃対象を選択して攻撃を行う
•	モンスターを破壊するとポイントを獲得する

３．システム上の工夫点
•	動的な名称生成: generateHuntingName 関数により、サイズ・属性・種族を組み合わせた多彩な名称を持つモンスターが登場する。
•	フィールド回転システム: rotateFields 関数による強制的な位置移動。前衛（攻撃）と後衛（待機・回復）の役割が毎ターン入れ替わるため、数ターン先を読んだ配置が不可欠となる。
•	状態異常の永続化: 一部のマイナスパワー（状態異常）は解除手段を用いない限り、対象が撃破されるまで継続する仕様となっている。
４．カードの種類と詳細
｛モンスターカード｝
•	攻撃に必要なパワー量を持つ
•	前衛にいる場合のみ攻撃可能
•	状態異常を付与する特殊モンスターも存在する

【状態異常（マイナスパワー）】
ターン終了時の「状態チェック」フェーズで効果が発揮される。（現在の予定）
•	アシッド・ポイズン: 毎ターン1ダメージを付与。
•	イルネス: 毎ターン2ダメージを付与する強力な病。
•	スリープ: 覚醒判定（コイントス）に成功するまで攻撃行動が不能となる。
【特殊能力モンスター】
•	コピー能力: 相手モンスターの攻撃力を模倣して戦う。
•	カリイジョウ: 直接ダメージではなく、攻撃によって「アシッド」等の状態異常を付与することに特化している。
【サポートカード】
•	ドロー系: 2枚〜4枚のカードを山札から引き、手札を補充する。
•	位置操作系: 「相手呼び出し」や「自分移動」により、フィールド上のモンスター配置を任意に入れ替える。
•	強化・回復系: パワーの付与や、永続する状態異常を解除する「コンディションヒール」
•	バフ・デバフ
•	手札入れ替えなどが存在する。
